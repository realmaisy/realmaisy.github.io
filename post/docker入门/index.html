1

# 安装命令
apt install docker.io

# 查看版本信息
docker -v
# Docker version 19.03.6, build 369ce74a3c

这个安装命令比较简单，但是版本可能不是最新的，区别不大。

基本操作

用nginx作为例子

拉取镜像

1
2

# :后面指定版本号，可以省略，默认拉取最新的
docker pull nginx:latest

查看现有镜像

1
2
3
4

docker images

REPOSITORY          TAG                 IMAGE ID            CREATED             SIZE
nginx               latest              6678c7c2e56c        2 weeks ago         127MB

运行镜像

-p 指定端口，左边是本地机器端口，右边是docker容器的端口，如下命令通过本机的IP:8080就能访问nginx页面
-d 指定容器在后台运行

1

docker run -p 8080:80 -d -it nginx

查看正在运行的容器

1
2
3
4

docker ps

CONTAINER ID        IMAGE               COMMAND                  CREATED             STATUS              PORTS                  NAMES
917fb9412c9c        nginx               "nginx -g 'daemon of…"   5 minutes ago       Up 5 minutes        0.0.0.0:8080->80/tcp   busy_shirley

进入容器

917是上面查询到的容器ID，输入部分即可

1
2
3

docker exec -it 917 /bin/bash

root@917fb9412c9c:/#

停止容器

1
2
3
4
5

docker stop 917

docker ps
CONTAINER ID        IMAGE               COMMAND             CREATED             STATUS              PORTS               NAMES

重启容器

docker start 加部分容器ID

1
2
3
4
5

docker start 917

docker ps
CONTAINER ID        IMAGE               COMMAND                  CREATED             STATUS              PORTS                  NAMES
917fb9412c9c        nginx               "nginx -g 'daemon of…"   30 minutes ago      Up 10 minutes       0.0.0.0:8080->80/tcp   busy_shirley

将容器打包成镜像

-a 作者
-m 说明文字
917 容器ID的部分
