Pip换源

技术
约 177 字 预计阅读 1 分钟 次阅读

文章目录

  • window
  • Ubuntu

window

在用户目录下C:\Users\maisy\新建文件夹pip ，然后在该文件夹下新建文件PIP.INI
完整路径：C:\Users\maisy\pip\PIP.INI
输入以下内容，换成阿里源

1
2

[global]
index-url = https://mirrors.aliyun.com/pypi/simple/

Ubuntu

在家目录下新建文件夹.PIP，然后在该文件夹下新建文件pip.conf
完整路径：/root/.pip/pip.conf
输入以下内容，换成阿里源

1
2

[global]
index-url = https://mirrors.aliyun.com/pypi/simple/
