Nginx

安装nginx

1

apt-get install nginx

反向代理配置，编辑文件/etc/nginx/sites-available/default，在里面添加如下内容；注意放在location /匹配访问根目录之前。

1
2
3

    location ^~ /gogogo/ {
        proxy_pass http://127.0.0.1:89/;
    }
  • 注意proxy_pass后面的地址，结尾带/表示重定向的时候不带匹配的地址；结尾不带/的话，会把匹配的地址也一起传过去。
  • 触发调用的webhooks地址，也要把后面的斜杠加上，例如：/gogogo/

重新加载配置

1

nginx -s reload

1

1
2
3
4

SET GOARCH=amd64

1

nohup ./AutoGit &
